# Holding Company of the Andes

A Next.js landing page for The Holding Company of the Andes.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

## Project Structure

- `app/page.tsx` - Main landing page with hero section
- `app/layout.tsx` - Root layout with Libre Baskerville font
- `public/logo.svg` - Company logo
- `public/background.jpg` - Background image (you'll need to add this)

## Note

You'll need to add the background image (`background.jpg`) to the `public` folder. The image should be the Andes mountain range photo.
